¿Que son y para que sirven los Selectores CSS?

Definen principalmente sobre qué elementos se utilizan un conjunto de reglas CSS como, por ejemplo.
Sintaxis: elthame Ejemplo: input, se aplica a cualquier elemento "input"
Selectores de clase:
En todos los elementos se tiene un atributo de class especificado. Sintaxis: .classname
Ejemplo: .index se selecciona cualquier elemento que tenga esta clase, “index”

¿Como se utilizan los selectores CSS?

Selector de ID:
Podemos seleccionar un elemento lo cual el valor de su atributo id. Solo puede haber un elemento determinado dentro de un documento.
Sintaxis: #idname Ejemeplo: #toc, se utiliza para cualquier elemento que tenga el ID “toc”
Selector universal:
Se selecciona todos los elementos. Opcionalmente esta restringido a un espacio de nombre específico o en los espacios de nombres.
Sintexis: *ns l * * l *¨ Ejemplo: * se aplica a todos los elementos del documento.

Otros selectores y combinadores

Selector de atributos:
Selecciona elementos extremadamente en el valor de un determinado.
Sintaxis: [attr] [attr=value] [attr-=value] [attr l =value] [attr^=value] [attr$=value] [attr*=value]
Ejemplo: [autoplay] seleccionará todos los elementos que pueden tener un atributo de “autoplay”.
Combinadores:
Combinador de hermanos adictivos. Los combinadores + seleccionan hermanos adictivos. Esto quiere decir que el segundo elemento sigue directamente al primero y ambos comparten el mismo elemento padre.
Sintaxis: A + B Ejemplo: La regla h2 + p se aplicará a todos los elementos "p" que siguen directamente a un elemento "h2" (en-US).
Combinador general de hermanos.
El combinador selecciona hermanos, ¿Qué quiere decir? Que el segundo elemento sigue al primero, ambos comparten el mismo elemento padre.
Sintaxis: A ~ B Ejemplo: La regla p ~span se aplicará a todos los elementos "span" que siguen un elemento "p".
